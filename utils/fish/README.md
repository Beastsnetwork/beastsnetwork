# Beastsnetwork Fish Shell Completions

This folder has basic Fish completions for `beastsnetworkd`, `beastsnetwork-wallet-cli`, and `beastsnetwork-wallet-rpc`. 

To use them, put those files (or symlink them) inside `~/.config/fish/completions/` or wherever your Fish completion files are.

See [https://fishshell.com/docs/current/completions.html#where-to-put-completions](https://fishshell.com/docs/current/completions.html#where-to-put-completions)
